Title : Portal Based Smart Billing System for Energy

The project is ongoing. We have completed basic study, started to build website and software part of the circuit.

The aim of the project is to develop a system that has facility to recharge the meter before consumption like “prepaid” and send a SMS alert to energy provider if someone try to tamper the system and disconnects the power supply in a case of non-payment of electricity bills.

Software tools used are Keil uvision Compiler, Proteus 8 Professional, Arduino IDE, VS Code, Brackets, Local Host, SQL Server.
